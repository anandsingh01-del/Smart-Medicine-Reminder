# Smart Medicine Reminder & Health Tracker

## 📌 Description

A Java desktop application that helps users manage medicines, receive reminders, and track health data like blood pressure and sugar levels.

---

## 🚀 Features

* User Login & Registration
* Add / View Medicines
* Medicine Reminder System
* Health Data Tracking
* Compliance Calculation
* Background Reminder using System Tray

---

## 🛠️ Technologies Used

* Java (Core Java)
* Java Swing (GUI)
* JDBC
* MySQL
* Multithreading

---

## 🗄️ Database Setup

1. Create database:

```sql
CREATE DATABASE health_tracker;
```

2. Import database:

```bash
mysql -u root -p health_tracker < backup.sql
```

---

## ▶️ How to Run

```bash
java -jar SmartMedicineTracker.jar
```

---

## 📂 Project Structure

* UI Layer (Swing)
* Service Layer (Logic)
* Database Layer (JDBC)
